Expert Glossary / 용어 사전 (Bilingual)

Sharpe Ratio / 샤프 지수
위험 한 단위당 초과 수익률. 높을수록 적은 변동성으로 큰 수익을 냈음을 의미합니다. (Risk-adjusted return per unit of volatility)
Sortino Ratio / 소르티노 지수
전체 변동성이 아닌 '하방 변동성(손실)'만을 고려한 위험 조정 수익률. (Downside-risk adjusted return)
IC (Information Coefficient)
예측값과 실제 수익률 사이의 상관계수. 모델의 '예측 숙련도'를 나타내며, 0.05 이상 시 유의미한 예측력이 있다고 봅니다. (Correlation between predicted and actual alpha)
MDD (Max Drawdown) / 최대 낙폭
특정 기간 고점 대비 얼마나 최대 하락했는지를 나타내는 리스크 지표. (Largest observed loss from peak to trough)
Information Ratio / 정보 비율
모델의 초과 수익이 얼마나 일관성 있게(낮은 변동성으로) 발생하는지를 측정합니다. (Consistency of excess returns)
Beta / 베타
시장(섹터) 전체 움직임에 대한 개별 모델의 민감도. 1보다 크면 시장보다 더 많이 움직입니다. (Sensitivity of strategy to market/benchmark)
