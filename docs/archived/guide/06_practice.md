# 챕터 6. 실습 및 확장 과제

## 6.1. 실습 목표
- 본 프로젝트의 각 모듈(수집, 전처리, 학습, 예측, 리포팅)을 직접 실행해보고, 데이터 흐름과 결과를 체험합니다.
- 실습을 통해 빅데이터 기반 AI 서비스의 전체 파이프라인을 이해하고, 현업 적용 역량을 기릅니다.

## 6.2. 실습 시나리오
1. **환경 구축:** Docker 및 의존성 설치, `.env` 설정
2. **과거 데이터 수집:** `collect_history.py` 실행 (2년치 데이터)
3. **초기 학습:** `initial_trainer.py`로 감성사전 및 모델 생성
4. **일일 운영:** 스케줄러를 통한 자동화 파이프라인 실행
5. **리포트 확인:** `output/` 폴더의 결과물 분석 및 대시보드 시각화

## 6.3. 확장 과제 예시
- **신조어/이슈 반영:** 사용자 사전(`user_dic.csv`)에 새로운 금융 신조어 추가 및 분석 정확도 비교
- **모델 개선:** Lasso 외 다른 ML 알고리즘(예: LightGBM) 적용 실험
- **데이터 시각화:** Streamlit 등으로 리포트 대시보드 구현
- **성능 튜닝:** Polars, Docker, DB 인덱싱 등 성능 최적화 실습
